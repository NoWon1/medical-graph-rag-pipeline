## 2024-05-24 - Avoid Alarming Colors for Standard Interactions
**Learning:** In medical applications, using alarming colors like red (`#e53935`) for standard interaction states (such as focus rings on text areas) falsely signals critical errors to users, causing unnecessary alarm and poor user experience.
**Action:** Rely on brand colors (e.g., green `#2fa36b`) for safe, expected UI feedback like focus rings. Only use red for actual errors or destructive actions.
