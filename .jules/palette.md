## 2026-07-21 - Replaced alarming red color for normal interaction state
**Learning:** In a medical application, alarming colors like red (`#e53935`) should not be used for standard interaction states such as focus rings. These colors falsely signal critical errors to users, creating unnecessary anxiety.
**Action:** Rely on brand colors (e.g., green `#2fa36b`) for safe, expected UI feedback like focus or active states.
